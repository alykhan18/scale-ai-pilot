# Scale100x WhatsApp Agent Panel

AI-powered WhatsApp sales operations dashboard built for the Scale100x.AI assignment.

## Features
- WhatsApp AI Agent Panel
- Human-in-the-Loop (HITL) Escalation Workflows
- Lead Intelligence Dashboard
- AI Resolution & Escalation Metrics
- Operational Activity Feed
- Responsive Enterprise SaaS UI

## Tech Stack
- React
- Tailwind CSS
- Lovable
- GitHub

## Assignment Focus
Built to demonstrate:
- Product thinking
- AI-native workflow design
- Operational UX systems
- Human-AI escalation architecture
- SaaS dashboard design and execution

## Key Workflow
AI handles lead qualification, conversation management, and automated follow-ups, while high-intent or sensitive conversations are escalated to human sales operators through the HITL escalation system.
